### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- Text Editor
- NodeJS
- PNPM

### Installation

1. Clone the repo
   ```sh
      https://github.com/maulanadityaa/todolist-react.git
   ```
2. Install package dependencies

   ```sh
     pnpm install
   ```

3. Run the app

   ```sh
     pnpm vite
   ```
